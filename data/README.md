# Data Directory

Этот каталог содержит файлы, которые будут загружены на файловую систему ESP32-S2 Mini.

## Загрузка файлов на ESP32

Для загрузки файлов на файловую систему ESP32 используйте следующую команду PlatformIO:

```bash
pio run --target uploadfs
```

Эта команда загрузит все файлы из директории `data` на файловую систему LittleFS ESP32.

## Файлы

- `year_schedule.json` - Файл с расписанием молитв на весь год. Содержит время молитв (Фаджр, Восход, Зухр, Аср, Магриб, Иша) для каждого дня года.

## Примечания

- Файловая система LittleFS имеет ограниченный размер (обычно 1-4 MB)
- Убедитесь, что файлы не превышают доступное пространство
- После изменения файлов в директории `data`, необходимо снова выполнить команду `pio run --target uploadfs`
